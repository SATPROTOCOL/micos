## 🌍 English

**micos** is a curated collection of AI agent workflow tools for systematic open-source project absorption, evaluation, and integration.

```
micos/
├── absorb-osp/          # 12-step closed-loop absorption workflow
│   ├── claude/          #   Claude Code integration
│   ├── hermes/          #   Hermes Agent integration
│   ├── templates/       #   Standardized output templates
│   └── shared/          #   Project indexes
└── (more tools coming)
```

### Quick Start

```bash
git clone https://github.com/SATPROTOCOL/micos.git
cd micos/absorb-osp
```

## 🌏 中文

**micos** 是一套 AI 智能体工作流工具集，专注于开源项目的系统性吸收、评估和融合。

每个子项目都经过：
- **安全审计优先** — 拒绝恶意代码，保护本地环境
- **深度评估吸收** — 理解架构、量化评分、同类合并
- **多框架兼容** — 支持 Claude Code、Hermes Agent 等主流框架

### 快速开始

```bash
git clone https://github.com/SATPROTOCOL/micos.git
cd micos/absorb-osp

# Claude Code 用户
cp -r claude/* ~/.claude/

# Hermes Agent 用户
cp -r hermes/* ~/.hermes/config/
```

[**探索 absorb-osp →**](./absorb-osp/)
